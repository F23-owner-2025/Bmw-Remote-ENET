# Installer

## End users

Do **not** use the scripts in this folder.

Download **`BMW-ENET-Setup.exe`** from the GitHub Releases page and choose Host or Client.

## Developers

- `Build-Windows.ps1` — build binaries + self-contained Setup.exe
- `uninstall.bat` — remove Host/Client services and Program Files folders
- `legacy/` — old bat/ps1 installers (kept for reference only)
